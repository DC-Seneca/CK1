# friendly-octo-spoon

A simple "Hello World" web page with a live clock showing Toronto time for students to know when to do their homework.

## Features

- Welcoming "Hello World" message for students
- Live clock displaying current time in Toronto timezone (EST/EDT)
- Current date display
- Responsive and visually appealing design
- Auto-updates every second

## How to Run

Simply open the `index.html` file in your web browser:

1. **Option 1**: Double-click the `index.html` file
2. **Option 2**: Right-click `index.html` and select "Open with" your preferred browser
3. **Option 3**: Drag and drop `index.html` into an open browser window

No server or additional setup required!

## For Students

The clock shows the current time in Toronto (Eastern Time Zone). Use this to:
- Know what time it is for your assignments
- Plan your homework schedule
- Stay synchronized with class timings

The page will automatically update the time every second, so you always have the current time displayed.
